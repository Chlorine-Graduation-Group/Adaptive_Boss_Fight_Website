Adaptive Boss Fight

An AI-driven combat experience that evolves with your playstyle.

View Source Code

Project Overview

Traditional boss fights often rely on static patterns that become repetitive. This project introduces an Adaptive Boss AI System that analyzes player behavior in real-time. By utilizing machine learning, the system classifies playstyles and dynamically adjusts the boss's tactics, creating a personalized and challenging gameplay experience.

Key Features

Adaptive AI Core

The system collects behavioral data (movement, dodge patterns, attack frequency) to classify the player events such as "Aggressive," "Defensive," or "Neutral," etc. adjusting the boss's phase mechanics accordingly.

AI Narrator

Powered by Large Language Models (LLMs), the narrator module provides real-time, context-aware commentary based on the player's actions and the current state of the battle.

Dynamic Music

An AI-generated music system that shifts intensity and style based on combat phases and player health, ensuring the audio atmosphere always matches the gameplay tension.
